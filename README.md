
## 电影速查  
基于mpvue 开发的一款小程序  
使用豆瓣api进行开发

## 主要功能  
* 最近热映电影展示
* 即将上映电影展示
* 电影搜索
* 显示电影详细信息及影评
* 电影各类排行榜
